Общие настройки

На данный момент в этой категории доступна лишь одна опция:

По умолчанию программа помещает все результаты расчетов в папку ".\results". При этом файлы с новыми результатами заменяют файлы со старыми результатами. После включения данной опции файлы каждого нового решения будут копироваться в новую папку с уникальным именем. Уникальное имя папки строится на основе параметров решаемой задачи следующим образом:

  • Символ 'b' или 's' в начале имени папки определяет тип краевых условий для задачи (s - боковой (side) подогрев, b - донный (bottom) подогрев);

  • Вещественное число, показывающее отношение высоты расчетной области к ее ширине;

  • Два вещественных числа с префиксами Pr и Gr показывают значения соответствующих безразмерных величин;

  • Четыре целых числа с префиксами t(op), l(eft), r(ight), b(ottom) показывают какие стенки являются жесткими, а какие - нет. 1 соответствует жесткое стенке, 0 - нежесткой стенке;

  • Два целых числа, разделенные символом 'x' - число узлов расчетной сетки по горизонтали и вертикали соответственно;

  • Три вещественных числа показывающие значения коэффициентов релаксации для температуры, функции тока и вихря соответственно;

  • Целое число показывающее максимальное число итераций.

Пример

Имя папки "s0,25_[Pr1;Gr100][t0;l1;r0;b1][100x25][0.3;0.3;0.3]_300" означает, что в этой папке находятся результаты для задачи бокового подогрева. Высота расчетной области в четыре раза меньше ее ширины. Число Прандтля равно 1, а Грасгофа - 100. Верхняя и правая стенки - не жесткие, а левая и нижняя - жесткие. Сетка имеет 100 узлов по горизонтали и 25 по вертикали. Коэффициенты релаксации для всех трех функций одинаковы и равны 0,3. Максимальное число итераций - 300.

 

Настройка графического движка

Данная вкладка позволяет выбрать графический движок программы. Смена движка произойдет только после перезапуска программы. Каждый из движков имеет свои уникальные настройки:

Параметр "Цветовая схема" в настройках OpenGL движка позволяет выбрать цветовую гамму, в которой будет нарисован результата вычислений. После смены цветовой схемы программа должна заново обработать результаты вычислений. На обработку данных необходимо некоторое время (это зависит как от мощности компьютера так и от размера сетки).
Для корректной работы Surfer движка ему необходимо указать путь к приложению Scripter.exe (входит в состав Surfer). Если путь указан неправильно, то обработка результатов произведена не будет и графическое представление результатов будет невозможно.
