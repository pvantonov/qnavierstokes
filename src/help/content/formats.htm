Все нижеперечисленные форматы данных являются текстовыми.

Файлы с решением задачи (T.dat, Psi.dat, Omega.dat, Vx.dat, Vy.dat)

Каждый файл содержит n строк по три вещественных числа в каждой строке. Первые два числа - это x и y координаты точки в двухмерном пространстве, а третье число - значение, которое принимает соответствующая функция в данной точке.

Файл и с информацией о конечноэлементной сетке (nvtr.dat)

Файл содержит n строк по три целых числа в каждой строке. Каждое из чисел - это номер точки расчетной области. Каждая тройка чисел определяет один треугольный конечный элемент с вершинами в указанных точках.

Файл с информацией о конечноразностной сетке (net.dat)

Файл содержит два целых числа (число узлов сетки по горизонтали и вертикали соответственно) и два вещественных числа (шаг сетки по горизонтали и вертикали соответственно).

Файлы с инструкциями для OpenGL (*.gl)

Данные файлы используются движком OpenGL для отрисовки изображения. В первой строчке этого файла находятся два вещественных числа - ширина и высота расчетной области соответственно (используется для задания масштаба рисунка). Во второй строчке содержится целое число (n), определяющее из скольких графических примитивов(треугольников) состоит рисунок.
Следом расположены n*3 строчки в каждой из которых два вещественных числа и три целых. Каждая тройка строчек определяет один графический примитив(треугольник). Первые два числа в строке - координаты вершины треугольника, а следующие три числа задают цвет этой вершины (в RGB формате).
Затем расположены m строчек по четыре вещественных числа. Они описывают сегменты изолиний. Каждая строчка определяет один сегмент(отрезок). Первые два числа - координаты начала сегмента, вторые два числа - координаты конца.
